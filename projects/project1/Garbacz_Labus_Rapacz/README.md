# Polska na torach

## Plakat:
![Plakat](Garbacz_Labus_Rapacz.png)

Plakat przedstawia kilka wizualizacji na temat polskiej kolei. W plakacie skupiliśmy się na 3 ważnych aspektach kolei: przewoźnikach kolejowych, opóźnieniach oraz infrastrukturze kolejowej. Ponadto plakat zawiera ciekawostki powiązane z polską koleją.
