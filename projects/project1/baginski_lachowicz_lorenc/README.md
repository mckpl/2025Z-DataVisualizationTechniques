# Czy warto studiować?

Plakat przedstawia analizę sytuacji osób z wykształceniem wyższym na rynku pracy. Celem plakatu jest ukazanie konsekwencji dynamicznie rosnącego udziału osób z wykształceniem wyższym w społeczeństwie.

Autorzy: Adam Bagiński, Leonard Lorenc, Szymon Lachowicz
